webbrowser
Open a Page
import webbrowser

# open page on default browser
webbrowser.open(
    'https://www.google.com/'
)

# open in a new window
webbrowser.open_new(
    'https://www.google.com/'
)

# open in a new tab
webbrowser.open_new_tab(
    'https://www.google.com/'
)
Use a Specific Browser
import webbrowser

b = webbrowser.get('chrome')
b.open('https://www.google.com/')
