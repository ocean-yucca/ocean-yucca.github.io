Terms
  • Node
    • Alias
      • Aliases in Neo4j name nodes, relationships, or expressions so you can refer to them later in the same query
    • Label, node type, for instance, Person, Movie
    • Property
      • id, unique, automatically assigned to node, not persistent, if a node is deleted, its ID may be reused
      • elementId, unique identifier for node
  • Relationship
    • Relationship Type, name or label assigned to a relationship
    • Property
      • id, a unique internal system-generated integer that identifies that specific relationship in the graph
      • elementId, globally unique string ID for relationship
